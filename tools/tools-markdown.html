何为Markdown语法

markdown语法，是一种纯文本格式的标记语言，让我们摆脱排版的困扰，专心写作。

Markdown IDE

推荐Typora这款IDE，优点如下：

  • 即时排版，不用预览
  • 支持各种插件，方便扩展
  • 快捷键方便齐全
  • 支持导出各种格式，包括word，pdf等。
  • 界面极其美观，各种主题可选

缺点，我认为除了不支持云笔记，好像没什么缺点。
