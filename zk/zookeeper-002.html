    1. 2.1. get节点内容

本文ZooKeeper版本：3.4.14，主要介绍zookeeper的常用命令。

服务端命令

进入bin目录：

  • ./zkServer.sh start，除了start还可以加{start|start-foreground|stop|restart|status|upgrade|print-cmd}。

    客户端命令

    连接服务端

  • /zkCli.sh 连接本地
  • ./zkCli.sh -server 192.168.1.1:2181 连接远程

    创建普通的节点

  • create /zhangjianbing "a boy"

    创建临时的节点

  • create -e /zhangjianbing "a boy"

    创建顺序节点

  • create -s /zhangjianbing "a boy"

    创建临时顺序节点

  • create -s -e /zhangjianbing "a boy"

    获取节点内容

  • get /zhangjianbing

    删除节点内容

  • delete /zhangjianbing 适用于该节点下没有子节点
  • rmr /zhangjianbing 递归删除，若该节点下有子节点，会先删除子节点

    修改节点

  • set /zhangjianbing

    在临时节点下是不能够创建持久节点的。

    get节点内容

    字段 内容 说明
    a boy 该节点绑定的值
    cZxid 0x262ea76 该节点创建时的id
    ctime Mon Jul 15 14:27:17 CST 2019 该节点的创建时间
    mZxid 0x262ea76 节点最新一次更新发生时的id
    mtime Mon Jul 15 14:27:17 CST 2019 该节点的修改时间
    pZxid 0x262ea76
    cversion 0 其子节点的更新次数
    dataVersion 0 该节点数据的更新次数
    aclVersion 0 节点ACL(授权信息)的更新次数
    ephemeralOwner 0x0 标识持久节点，若非，则为临时节点的sessionId
    dataLength 5 节点数据的字节数
    numChildren 0 子节点个数
